S905X Android Compile environment to build

Ready to compile

Compiling Android requires high machine configuration:

  • 64-bit CPU
  • 32GB physical memory + swap memory
  • 500GB of free disk space

The official recommendation is the Ubuntu 20.04 operating system.

The initialization of the compilation environment can refer to http://source.android.com/source/initializing.html 。

Ubuntu 18.04 and 20.04 package installation reference:

$ sudo apt-get update
$ sudo apt-get install curl python git
$ sudo apt-get install openjdk-8-jdk

Install Extra Toolchains For The Amlogic Platform

  • The Amlogic Platform requires extra toolchains for cross-compiling, you will need to follow these steps to setup:

Install Cross Compiler for U-Boot BL:

sudo chmod 777 /opt/
tar -zxvf CodeSourcery.tar.gz -C /opt
tar -xvf gcc-linaro-aarch64-none-elf-4.8-2013.11_linux.tar -C /opt
tar -zxvf gnutools.tar.gz -C /opt
tar -zxvf arc-4.8-amlogic-20130904-r2.tar.gz -C /opt

Install Cross-Compiler for Linux Kernel:

tar -xvf gcc-linaro-aarch64-linux-gnu-4.9-2014.09_linux.tar -C /opt
tar -zxvf gcc-linaro-arm-linux-gnueabihf.tar.gz -C /opt

Set Env for build

chmod u+x TOOLSENV.sh
sudo cp TOOLSENV.sh -P /etc/profile.d

Build Instrections

source build/envsetup.sh
lunch ohm-userdebug
# build kernel
./mk ohm -v 5.4 -t userdebug
# build android
make -j64
out/target/product/ohm/aml_upgrade_package.img
